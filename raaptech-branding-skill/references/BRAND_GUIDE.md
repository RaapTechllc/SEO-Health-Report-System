# RaapTech Brand Guide

Visual identity standards for all RaapTech materials.

---

## Brand Essence

**Who We Are**: Chicago-based automation and systems company  
**What We Do**: Help founders build businesses that don't need them  
**How We Sound**: Human, strategic, urgent (without sounding like AI)

---

## Visual Identity

### Logo

**Primary Mark**: `assets/raaptech_logo.png`
- Black and white design
- Chicago skyline integrated into wordmark
- Bold, custom typography
- Dynamic swoosh element

**Usage Rules**:
- Always use provided logo file (never recreate)
- Minimum size: 1 inch wide (print) or 100px wide (digital)
- Clear space: Equal to height of "R" on all sides
- Place on cover page, header, or footer
- Never: Rotate, stretch, recolor, add effects

**Placement**:
- Documents: Cover page (centered) or header (left-aligned)
- Digital: Header or footer, subtle presence
- Presentations: Title slide and footer on all slides

### Color Palette

| Color | HEX | Usage |
|-------|-----|-------|
| **Black** | #000000 | Primary text, headers, logo |
| **White** | #FFFFFF | Backgrounds, negative space |
| **Gray-1** | #333333 | Secondary text |
| **Gray-2** | #666666 | Tertiary text, captions |
| **Gray-3** | #CCCCCC | Borders, dividers |

**Rules**:
- Dominant: Black text on white backgrounds
- Accent: Gray for hierarchy and separation
- Never use other colors without explicit approval
- High contrast required for all text (WCAG AA minimum)

### Typography

**Headers**:
- Font: Bold sans-serif (Helvetica Bold, Arial Black, or similar)
- H1: 24-28pt, all caps or title case
- H2: 18-22pt, title case
- H3: 14-16pt, title case

**Body**:
- Font: Clean sans-serif (Helvetica, Arial, or similar)
- Size: 11-12pt
- Line spacing: 1.5x
- Paragraph spacing: 6pt after

**Emphasis**:
- Bold for strong emphasis only
- Italic for subtle emphasis or quotes
- Never underline (except hyperlinks)

**Rules**:
- Consistent hierarchy (don't skip heading levels)
- Max 2 font families per document
- Minimum 11pt for body text
- No decorative or script fonts

### Layout Standards

**Margins**:
- Standard documents: 1 inch all sides
- Proposals: 1.25 inch left, 1 inch others
- One-pagers: 0.75 inch all sides (maximize space)

**Whitespace**:
- Strong whitespace between sections
- At least 12pt space between paragraphs
- At least 18pt space before headers
- Never cram content

**Hierarchy**:
- Clear H1 → H2 → H3 structure
- Visual distinction between levels (size + weight)
- Consistent spacing throughout

**Components**:

**Tables**:
- Header row: Black background (#000000), white text (#FFFFFF)
- Data rows: White background, black text
- Borders: 1pt solid, Gray-3 (#CCCCCC)
- Padding: 8pt cell padding
- No zebra striping (alternating colors)

**Callout Blocks**:
- Background: Gray-3 (#CCCCCC) or White with Gray border
- Text: Black, bold for emphasis
- Padding: 12pt all sides
- Use for: Key insights, important notes, warnings

**Lists**:
- Bullet style: Simple circles or squares
- Indent: 0.25 inch from margin
- Spacing: 3pt between items
- Use for: 3+ related items, features, benefits

**Images/Graphics**:
- Black and white or grayscale preferred
- High contrast required
- Captions: Gray-2, 10pt, below image
- Placement: Never interrupt reading flow

## Document Types

### Proposals

**Cover Page**:
- Logo: Centered, 2-3 inches wide
- Title: 36pt, bold, centered
- Subtitle: 18pt, centered
- Date: 12pt, centered, bottom third

**Interior Pages**:
- Header: Logo small (left), page number (right)
- Footer: Company info (left), date (right)
- Margins: 1.25" left, 1" others

### One-Pagers

**Layout**:
- Logo: Top left, 1 inch wide
- Title: Top center or left, 24pt bold
- Content: 2-3 columns if needed
- Margins: 0.75" all sides
- No footer (maximize space)

**Content Priority**:
1. Hook (biggest, boldest)
2. Problem/Insight
3. Mechanism
4. Proof
5. CTA

### Reports

**Cover**:
- Logo: Top left or centered
- Title: 28pt, bold
- Author/Date: 12pt, subtitle area

**Interior**:
- Header: Section name (left), page number (right)
- Footer: Confidential notice if needed
- Clear section breaks (page breaks or heavy dividers)

### Emails

**Signature**:
```
Kyle Raap
RaapTech LLC
[Logo small, if possible]
```

**Format**:
- Plain text preferred (maximum compatibility)
- HTML if needed: Black text, white background, simple layout
- No backgrounds, no colors (except logo)

## Print Specifications

**Business Cards**:
- Size: 3.5" × 2"
- Logo: Front, centered or left
- Info: Name, title, email, phone
- Back: Optional tagline or QR code

**Letterhead**:
- Logo: Top left, 1.5" wide
- Company info: Footer, 9pt
- Paper: White, 24lb minimum

---

## Digital Standards

**Website/Landing Pages**:
- Logo: Header, left-aligned, 150px wide
- Color scheme: Black text, white backgrounds, gray accents
- Typography: System fonts for speed (Helvetica, Arial fallback)

**Social Media**:
- Logo: Profile image, square crop
- Cover: Minimal, black/white Chicago skyline if needed
- Posts: Black text, white backgrounds, high contrast

**Presentations**:
- Title slide: Logo centered
- Content slides: Logo footer, small
- Backgrounds: White or very light gray
- Text: Black, minimum 18pt

---

## Brand Checklist

Before finalizing any branded material:

- [ ] Logo used correctly (size, placement, clearspace)
- [ ] Only black/white/gray colors used
- [ ] Typography consistent (bold headers, clean body)
- [ ] Strong whitespace maintained
- [ ] Clear hierarchy (H1 → H2 → H3)
- [ ] High contrast (all text readable)
- [ ] Tables formatted correctly (black header, gray borders)
- [ ] Callout blocks used for key insights
- [ ] No decorative fonts
- [ ] Clean, professional aesthetic
- [ ] Chicago identity present (logo shows skyline)
