## MODULE 2: FRAMEWORK MASTERY (Weeks 3-4)

### Exercise 2.1: Write Using AIDA

**Choose a product or service YOU know well.**

Write copy following strict AIDA:

```
ATTENTION (1-2 sentences):
[Interrupt pattern, create curiosity, appeal to self-interest]

INTEREST (3-4 sentences):
[Provide facts, engage logical mind, show it's relevant]

DESIRE (2-3 sentences):
[Paint after-picture, appeal to emotion, show transformation]

ACTION (1-2 sentences):
[Clear next step, remove friction, create gentle urgency]
```

**Then evaluate:**
- [ ] Does attention actually stop me?
- [ ] Do I understand why it's relevant?
- [ ] Do I actually want it?
- [ ] Is action clear and easy?

---

### Exercise 2.2: Write Using PAS

**Use the same product/service.**

Write copy following PAS:

```
PROBLEM (2-3 sentences):
[Identify the pain point specifically]

AGITATION (2-3 sentences):
[Make it feel urgent, show consequences, create emotional weight]

SOLUTION (2-3 sentences):
[Present your answer, show mechanism, show results]
```

**Then compare to AIDA:**
- Which version creates more urgency?
- Which version creates more desire?
- Which version feels more honest?
- Which would you actually use?

---

### Exercise 2.3: Write Using Hook-Story-Offer

**Use the same product/service.**

Write copy following Hook-Story-Offer:

```
HOOK (1 sentence):
[Competence / Curiosity / Connection]

STORY (3-4 sentences):
[Problem → Insight → Solution, building narrative]

OFFER (1-2 sentences):
[Outcome-focused ask, clear next step]
```

**Comparison:**
```
AIDA is: Attention → Interest → Desire → Action
PAS is: Problem → Agitation → Solution
HSO is: Hook → Story → Offer

When would I use each?
AIDA: _____________________
PAS: _____________________
HSO: _____________________
```

## MASTERY INDICATORS

### You're at Level 1 (Formula Follower) When:
- [ ] You can identify which framework a piece of copy uses
- [ ] You can write basic AIDA copy
- [ ] You understand each master's core principle

### You're at Level 2 (Framework Adapter) When:
- [ ] You can choose the right framework for a situation
- [ ] You can write using PAS, HSO, and AIDA fluently
- [ ] You can explain why one master would approach differently than another

### You're at Level 3 (Tone Composer) When:
- [ ] You can blend multiple tones intentionally
- [ ] You understand the psychology behind each principle
- [ ] You can write copy that hits multiple Cialdini principles at once

### You're at Level 4 (Insight Generator) When:
- [ ] You generate unique insights for your market
- [ ] Your copy reframes how people think
- [ ] You've developed your own voice that blends master principles

### You're at Level 5 (Master Persuader) When:
- [ ] You move people through belief layers naturally
- [ ] Your copy feels human and strategic simultaneously
- [ ] People remember your copy and repeat your insights
