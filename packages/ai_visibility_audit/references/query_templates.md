# AI Visibility Test Query Templates

## Overview

Use these templates to generate comprehensive test queries for AI visibility audits. Replace placeholders with actual brand/product information.

---

## Query Categories

### 1. Brand Awareness Queries

Direct queries about the brand itself.

```
What is {brand_name}?
Tell me about {brand_name}
{brand_name} company overview
Who founded {brand_name}?
What does {brand_name} do?
{brand_name} history
Where is {brand_name} headquartered?
{brand_name} company information
```

**Expected outcome:** AI should know the brand exists and provide accurate basic information.

---

### 2. Product/Service Category Queries

Queries where brand should appear as an option.

```
Best {product_category} options
Top {product_category} providers
What is the best {product_category}?
{product_category} recommendations
Top 10 {product_category} companies
Best {product_category} for {use_case}
{product_category} market leaders
Who makes the best {product_category}?
```

**Expected outcome:** Brand should appear in lists of recommended options.

---

### 3. Problem/Solution Queries

Queries about problems the brand solves.

```
How to {solve_problem}
{problem} solutions
Best way to {achieve_goal}
How to choose a {product_category}
What to look for in {product_category}
{pain_point} help
How do I {task}?
{industry} best practices
```

**Expected outcome:** Brand should be mentioned as a solution.

---

### 4. Comparison Queries

Direct comparisons with competitors.

```
{brand_name} vs {competitor}
Compare {brand_name} and {competitor}
Is {brand_name} better than {competitor}?
{brand_name} or {competitor}?
{competitor} alternatives
{brand_name} competitors
{brand_name} vs {competitor} vs {competitor_2}
```

**Expected outcome:** Brand should be represented fairly with accurate differentiators.

---

### 5. Reputation Queries

Queries about brand perception.

```
{brand_name} reviews
Is {brand_name} good?
{brand_name} complaints
{brand_name} customer experience
{brand_name} trustworthy?
{brand_name} quality
{brand_name} reputation
Should I use {brand_name}?
{brand_name} pros and cons
```

**Expected outcome:** AI should reflect genuine sentiment without hallucinated issues.

---

### 6. Expertise Queries

Queries where brand expertise should surface.

```
{industry} trends
{topic} best practices
How does {technology} work?
{industry} statistics
{topic} guide
{industry} expert opinions
```

**Expected outcome:** Brand's content or experts should be cited if authoritative.

---

### 7. Local/Specific Queries

Location or context-specific queries.

```
Best {product_category} in {location}
{brand_name} {location}
{product_category} near me
{industry} companies in {location}
```

**Expected outcome:** Brand should appear for relevant geographic queries.

---

## Query Generation Strategy

### Minimum Query Set (10 queries)

1. Brand awareness: 2 queries
2. Product category: 3 queries
3. Problem/solution: 2 queries
4. Comparison: 2 queries
5. Reputation: 1 query

### Standard Query Set (20 queries)

1. Brand awareness: 3 queries
2. Product category: 5 queries
3. Problem/solution: 4 queries
4. Comparison: 4 queries
5. Reputation: 3 queries
6. Expertise: 1 query

### Comprehensive Query Set (40+ queries)

Include all categories with multiple variations, plus:
- Long-tail queries
- Question variations (what, how, why, when, where)
- Negative queries (problems with, issues, complaints)
- Future-oriented (upcoming, trends, predictions)

---

## Query Customization Tips

### For B2B Companies
- Focus on industry-specific terminology
- Include role-based queries ("best {product} for {role}")
- Add enterprise/scale qualifiers

### For B2C Companies
- Include price-sensitivity queries
- Add user-type variations ("best for beginners")
- Include emotional queries ("easy to use", "trusted")

### For Local Businesses
- Heavy emphasis on location queries
- Include "near me" variations
- Add service area coverage queries

### For SaaS Products
- Include integration queries ("{brand} + {other tool}")
- Add feature-specific queries
- Include pricing/plan queries

## Response Analysis Checklist

For each query response, evaluate:

- [ ] Brand mentioned? (yes/no)
- [ ] Position of mention (first/middle/last/not mentioned)
- [ ] Accuracy of information
- [ ] Sentiment (positive/neutral/negative)
- [ ] Competitors mentioned
- [ ] Citations or sources provided
- [ ] Recency of information
- [ ] Completeness of response
