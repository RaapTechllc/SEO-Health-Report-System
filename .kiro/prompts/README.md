# Prompts

Reusable prompts for common workflows. Use with `@prompt-name`.

## Core Workflow
- `plan-feature` - Start spec-driven development
- `execute` - Execute implementation tasks
- `next-task` - Continue to next task
- `workflow-status` - Check active specs

## Quality
- `code-review` - Review code changes
- `security-audit` - OWASP security audit
- `a11y-audit` - Accessibility audit
- `test-coverage` - Analyze test coverage

## Project
- `quickstart` - Bootstrap new project
- `prime` - Load project context
- `deploy-checklist` - Pre-deployment checks

## Reflection & Learning
- `self-reflect` - Agent self-improvement (RBT)
- `reflect` - Capture session learnings to LEARNINGS.md
- `rca` - Root cause analysis
